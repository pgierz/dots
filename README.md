# Dots
These are my dotfiles

## What you get

The following is included:
+ `vimrc`: A plugin based vim configuration. I could probably do more with this, but keeping it minimal is nice too... If you're not using vim, you should be. Spend some time learning it, it'll make your life easier.
+ `zshrc`: A very minimally changed [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh) rc file. It gives you slurm control via `sq`, which gives you a "just you" view of any slurm queue. I'm considering adding a few files to better deal with "useful" defaults for simulation based climate research. 
+ `dir-colors`: I use [nord](https://github.com/arcticicestudio/nord). It's pretty
+ `tmux.conf`: A vim-esc tmux configuration. I really need to learn how to configure tmux correctly. I would probably be happier if I figure it out.
